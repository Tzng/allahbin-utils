## 特性

- TypeScript 类型完整
- Node + Browser 双端支持
- 按需导入与多入口
- 完整测试与覆盖率

## 快速开始

```bash
pnpm add allahbin-utils
```

```ts
import { capitalize } from 'allahbin-utils';
console.log(capitalize('hello world'));
```
